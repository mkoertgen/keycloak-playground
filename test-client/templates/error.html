Troubleshooting

  • Make sure Keycloak is running at https://keycloak.localhost
  • Verify the realm demo exists in Keycloak
  • Check that the client demo-app is configured correctly:
    • Valid Redirect URIs: https://app.localhost/*
    • Web Origins: https://app.localhost
    • Access Type: confidential
  • Ensure .env file has correct OIDC_CLIENT_SECRET
  • Check /etc/hosts contains: 
    127.0.0.1 keycloak.localhost app.localhost
